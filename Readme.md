# 🛒 E-Commerce Sales Analysis

An end-to-end Data Analytics project built using the Olist Brazilian E-Commerce dataset. This project demonstrates the complete data analytics workflow, including Data Cleaning, Exploratory Data Analysis (EDA), SQL Analysis, Excel Dashboard, and Power BI Dashboard to generate actionable business insights.

---

# 📌 Project Overview

The objective of this project is to analyze e-commerce sales data and identify business trends related to revenue, customer behavior, product performance, seller performance, and customer satisfaction.

The project covers the complete analytics lifecycle from raw data processing to interactive dashboard development.

---

# 🎯 Objectives

- Perform data cleaning using Python.
- Conduct Exploratory Data Analysis (EDA).
- Analyze business problems using SQL.
- Build an interactive Excel Dashboard.
- Develop a Power BI Dashboard.
- Generate business insights and recommendations.

# 💡 Key Business Insights

- Health & Beauty generated the highest revenue.
- November 2017 recorded the highest monthly sales.
- Most customer reviews are 5-star.
- Revenue is concentrated in a few major states.
- Average Order Value provides insights into customer purchasing behavior.

---

# 🚀 Business Recommendations

- Increase inventory before peak sales periods.
- Focus marketing on high-performing product categories.
- Improve delivery performance to enhance customer satisfaction.
- Expand operations in high-revenue regions.
- Strengthen customer loyalty programs.
